# Менеджер задач (Task Manager)

Простой консольный менеджер задач на языке Go.  
Позволяет создавать задачи, просматривать список, менять статус, выводить и удалять задачу по `Id`.  
Данные хранятся в файле `task.json` в корне проекта.

## Требования

- Go 1.25 и выше  
- Зависимость: [`github.com/fatih/color`](https://github.com/fatih/color)

## Установка и сборка

Склонируйте репозиторий и выполните сборку:

```sh
git clone https://github.com/BKBTVETH98/task-manager-CLI
cd "Task manager GO"
go mod tidy    # установить зависимости
go build ./... # собрать все пакеты, в том числе main
```

В результате появится исполняемый файл в текущей директории. Его можно запускать напрямую:

```sh
./main.exe    # на Windows
./main        # на других ОС
```

## Структура проекта

```
.
├── actionFile/   # обработчики действий (создать, вывести, изменить и т.д.)
│   └── action.go
├── main/         # точка входа
│   └── main.go
├── task/         # доменные сущности и работа с файлом
│   ├── file.go
│   ├── task.go
│   └── vault.go
├── task.json     # файл хранения данных (создаётся автоматически)
├── go.mod
└── .gitignore
```

### Важные пакеты

- [`task.NewTask`](task/task.go) – создание новой задачи.
- [`task.Vault`](task/vault.go) – хранилище задач, чтение/запись в `task.json`.
- [`task.GetReader`](task/file.go) – чтение строки из `stdin`.
- [`actionfile`](actionFile/action.go) – функции команд (Newtask, GetTask, FoundTaskId, ViewTaskId, DelTaskId).

## Использование

При запуске программа выводит меню:

```
выберите действие, введя цифру от 1 до 5
1, Создать таску
2, вывести все таски
3, изменить статус таски по Id
4, вывести таску по Id
5, удалить таску по Id

Ваш выбор:
```

- **1** – создать новую задачу (описание и статус: `running`, `pause`, `done`).
- **2** – вывести все задачи (форматированный JSON).
- **3** – изменить статус задачи по `Id`.
- **4** – показать задачу по `Id`.
- **5** – удалить задачу по `Id`.


После каждого изменения файл `task.json` перезаписывается.
